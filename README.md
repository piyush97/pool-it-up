## Run Locally

Clone the project

```bash
  git clone https://github.com/piyush97/pool-it-up
```

Go to the project directory

```bash
  cd pool-it-up
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** React Native, Redux, TailwindCSS
